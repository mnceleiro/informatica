    1. Virtualización y contenedores

    Existen herramientas que permiten, dentro de tu sistema Windows o Linux, virtualizar otro ordenador. Esto es, dentro de tu ordenador tener "varios ordenadores" en entornos totalmente aislados.
    Estos ordenadores aislados se denominan máquinas virtuales y el proceso se denomina virtualización. Algunas herramientas de virtualización son las siguientes:

    • VirtualBox (Código abierto)
    • VMWare
    • Hyper-V (viene con las versiones profesionales de Windows).
