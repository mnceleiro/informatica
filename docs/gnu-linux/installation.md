Existen multitud de tutoriales de instalación y configuración de sistemas operativos. En este caso iré recomendando uno o varios en cada sección para que podáis seguirlos (en lugar de repetir esos tutoriales yo mismo). 

## Instalación de Ubuntu 24.04
En estos dos enlaces se explica como instalarlo tanto con particionado manual: 

- [Enlace 1](https://ubuntuhandbook.org/index.php/2024/04/install-ubuntu-24-04-desktop/) 
- [Enlace 2](https://www.linuxtechi.com/install-ubuntu-24-04-step-by-step/).

## Actualización de paquetes
Una vez arrancado actualiza el software con los siguientes comandos

```
    sudo apt update
    sudo apt full-upgrade
```

## Instalación de los guest additions en VirtualBox
Esto añade algunas funcionalidades: posibilidad de pantalla completa, copiar y pegar desde tu Windows 11 y la máquina virtual, etc.

Para instalar los guest additions puedes seguir [este manual](https://www.linuxtechi.com/install-ubuntu-24-04-step-by-step/).

!!! Nota
    Si no te funciona: apaga y enciende de nuevo la máquina y, una vez encendida e identificado en ella, modifica el tamaño de la ventana (restaura y maximiza la ventana).
